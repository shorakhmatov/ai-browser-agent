# Быстрый старт

## 1. Установка (2 минуты)

```bash
# Клонируем репозиторий
git clone https://github.com/yourusername/ai-browser-agent.git
cd ai-browser-agent

# Устанавливаем зависимости
pip install -r requirements.txt

# Устанавливаем браузер
playwright install
```

## 2. Настройка (1 минута)

```bash
# Копируем пример конфигурации
cp .env.example .env

# Открываем .env и добавляем API ключ
# ANTHROPIC_API_KEY=sk-ant-...
```

## 3. Запуск (1 минута)

### Интерактивный режим
```bash
python cli.py
```

Затем в терминале:
```
> task найди цену товара на amazon.com
```

### Одна задача
```bash
python cli.py "перейди на github.com и найди trending repositories"
```

## Примеры задач

```
"Заполни форму регистрации на примере.com"
"Найди контактную информацию на сайте"
"Добавь товар в корзину и оформи заказ"
"Скопируй информацию о продукте"
"Найди и скачай файл"
```

## Что происходит

1. Браузер открывается (видимый Chromium)
2. Агент анализирует страницу
3. Выполняет действия (клики, ввод текста, прокрутка)
4. Показывает результаты в терминале
5. Запрашивает подтверждение для опасных действий

## Горячие клавиши

- `Ctrl+C` - остановить агента
- `y` - одобрить деструктивное действие
- `n` - отклонить деструктивное действие

## Решение проблем

**Браузер не открывается:**
```bash
playwright install chromium
```

**API ключ не работает:**
- Проверьте, что ключ в .env файле
- Убедитесь, что используется правильный ключ Anthropic

**Агент зависает:**
- Нажмите Ctrl+C
- Проверьте интернет соединение
- Попробуйте более простую задачу
